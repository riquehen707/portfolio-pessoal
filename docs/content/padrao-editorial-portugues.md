# Padrão Editorial em Português

## Voz

A voz do projeto é direta, clara, estratégica e sóbria. O texto deve parecer uma leitura editorial de negócio, não uma peça publicitária.

O conteúdo precisa:

- partir de um problema real;
- explicar o mecanismo por trás do problema;
- mostrar o benefício prático;
- indicar um próximo passo possível.

## Ritmo

Use frases curtas e médias. Prefira verbos concretos: organizar, revisar, reduzir, transformar, comparar, medir, ajustar, conduzir.

Evite empilhar adjetivos. Quando uma frase depender de palavras como "inovador", "completo", "estratégico" ou "resultado", revise para explicar o motivo.

## Padrões úteis

- Antes de investir, revise a base que recebe o contato.
- Organize presença, oferta e atendimento para transformar interesse em ação.
- Use conteúdo para reduzir dúvida, não apenas para aparecer.
- Nem todo problema de captação se resolve com mais verba.

## Evitar

- Promessas absolutas.
- Tom de agência com excesso de entusiasmo.
- Slogans como argumento.
- Frases genéricas sobre crescimento, inovação ou resultado sem explicar o caminho.
- Repetir "clareza" quando houver uma palavra mais específica: oferta, agenda, página, mensagem, recorte, fluxo, atendimento ou métrica.

## Uso da assinatura

"Clareza gera resultado." continua sendo uma assinatura curta da marca. Use com moderação em peças institucionais, como imagem OG ou assinatura visual. Não usar como título recorrente, subtítulo de seção ou fechamento automático de artigo.
