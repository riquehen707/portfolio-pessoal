# Modelo de entrada do artigo-matriz

Copie o bloco abaixo e preencha. Apenas `linhaEditorial` e `textoBruto` são obrigatórios.

```yaml
linhaEditorial:
tituloProvisorio:
objetivo:
publico:
ideiaCentral:
cta:
restricoes:
```

## Texto bruto

```text
Cole aqui o texto original.
```

## Observações opcionais

```text
Indique fontes já usadas, experiências que não podem ser removidas, links relevantes, prazo ou qualquer dúvida que deva permanecer explícita.
```

As regras de interpretação estão em [`03-producao/entrada-artigo-matriz.md`](../03-producao/entrada-artigo-matriz.md).
