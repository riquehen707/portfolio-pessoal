## Visão geral

`Loja Kitsune` é um estudo de caso criado para simular uma loja de nicho com catálogo editorial forte, páginas de categoria bem resolvidas e foco em busca orgânica.

A proposta foi montar um case honesto para representar meu trabalho com `SEO` para e-commerce, sem inventar cliente real.

## Objetivo

O projeto foi desenhado para validar uma arquitetura de loja que prioriza:

- leitura fácil no mobile
- hierarquia visual de produto
- indexação limpa para categorias e coleções
- página de produto com menos atrito

## Decisões principais

Tratei a camada de produto como uma combinação entre vitrine e conteúdo:

- páginas com textos de apoio para SEO
- filtros e categorias mais previsíveis
- imagens e CTA organizados por peso real na compra
- estrutura preparada para rich snippets e metadados

## Stack e arquitetura

Usei `Next.js` para a camada de experiência, `Shopify` como referência de operação de catálogo e `SCSS` para manter controle fino sobre responsividade e ritmo visual.

## O que esse case mostra

Mesmo sendo um estudo, ele mostra bem a minha forma de pensar:

- SEO técnico como parte do produto
- interface leve para loja virtual
- performance como fator de conversao

## Próximo passo

Esse estudo serve como base para futuros cases reais em e-commerce, principalmente em lojas de nicho e catálogos com potencial orgânico.
