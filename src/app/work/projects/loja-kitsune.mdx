## Visao geral

`Loja Kitsune` e um estudo de caso criado para simular uma loja de nicho com catalogo editorial forte, paginas de categoria bem estruturadas e foco em busca organica.

O objetivo foi preencher o portfolio com um caso que represente meu trabalho em `SEO` para e-commerce sem fingir um cliente real.

## Objetivo

O projeto foi desenhado para validar uma arquitetura de loja que prioriza:

- leitura facil no mobile
- hierarquia visual de produto
- indexacao limpa para categorias e colecoes
- pagina de produto com menos atrito

## Decisoes principais

Tratei a camada de produto como uma combinacao entre vitrine e conteudo:

- paginas com textos de apoio para SEO
- filtros e categorias mais previsiveis
- imagens e CTA organizados por peso real na compra
- estrutura preparada para rich snippets e metadados

## Stack e arquitetura

Usei `Next.js` para a camada de experiencia, `Shopify` como referencia de operacao de catalogo e `SCSS` para manter controle fino sobre responsividade e ritmo visual.

## O que esse case mostra

Mesmo sendo um estudo, ele comunica bem minha forma de pensar:

- SEO tecnico como parte do produto
- interface leve para loja virtual
- performance como fator de conversao

## Proximo passo

Esse estudo serve como base para futuros cases de clientes reais em e-commerce, principalmente em lojas de nicho e catalogos com potencial organico.
